# Seamless.ly

## A/B Testing with Vanity and Split

This is a sample app to test out these gems, with just a sign up page.


### with Vanity

#### Start up
  * brew install redis
  * clone/fork
  * bundle
  * rake db:create
  * rake db:migrate
  * start redis server
    * ```` redis-server /usr/local/etc/redis.conf ````
  * start rails server
    * ```` rails s ````
